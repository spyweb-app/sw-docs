## Features

<CardGrid stagger>
	<Card title="Zero Dependencies" icon="open-book">
		~7MB self-contained binary. Completely portable, no runtime required.
	</Card>
	<Card title="Lua Scripting" icon="pencil">
		10 hook stages plus persistent storage for counters, cursors, and shared state.
	</Card>
	<Card title="Hot Reload" icon="random">
		Save a config or Lua script and SpyWeb respawns the job instantly.
	</Card>
	<Card title="Browser Automation" icon="laptop">
		Built-in CDP support for Chrome, Edge, or Brave. Scrape JS-rendered pages.
	</Card>
	<Card title="Multi-Worker" icon="rocket">
		Per-job concurrency with URL queue mode and shared Lua state.
	</Card>
	<Card title="Dual Binary" icon="setting">
		Choice of headless CLI or silent system tray for background runs.
	</Card>
</CardGrid>

## Quick Start

<Tabs>
  <TabItem label="config.toml">
```toml
name = "HN Front Page"
url = "https://news.ycombinator.com"
selector = ".athing"
fields = ["title:.titleline > a", "link:.titleline > a@href"]
keywords = ["rust", "linux", "open source"]
```
  </TabItem>
  <TabItem label="hooks.lua">
```lua
function before_fetch(request, ctx)
    print("Fetching: " .. request.url)
    return request
end
```
  </TabItem>
  <TabItem label="defer.lua">
```lua
function on_finally(ctx)
    print("Cycle finished")
end
```
  </TabItem>
</Tabs>

```bash
spyweb start
```

See [Job Configuration](/job-configuration/toml-config) for all options, or [Get Started](/getting-started) for download commands.
